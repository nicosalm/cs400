How to use Bogosort

To use Bogosort, you must first have an array of numbers. Then, you must shuffle the array until it is sorted. If you are lucky, it will be sorted in the first shuffle. If you are unlucky, it will take a long time. But, if you are really unlucky, it will never be sorted. This is why it is so good.

  1. Shuffle the array.
  2. Check if the array is sorted.
  3. If the array is sorted, stop. If not, go to step 1.
  4. Profit.
